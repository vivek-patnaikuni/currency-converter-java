# Currency Converter Application (Java)

A desktop-based Currency Converter application developed using Core Java and Swing.

## Features
- Real-time currency conversion logic
- User-friendly graphical interface (Swing)
- Input validation and error handling
- Object-Oriented Programming implementation
- Modular class structure

## Technologies Used
- Java
- Swing (GUI)
- OOP Concepts
